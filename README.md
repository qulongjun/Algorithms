### 算法范式

算法范式是基于类的设计的通用方法或方法的算法。 这是一个比算法概念更高的抽象, 就像一个
算法是比计算机程序更高的抽象。

`A` - 初学者, `B` - 进阶, `C` - 高级, `D` - 困难, `E` - 挑战
